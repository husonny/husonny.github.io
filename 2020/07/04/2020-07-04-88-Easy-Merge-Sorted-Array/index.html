[88] Merge Sorted Array

Given two sorted integer arrays nums1 and nums2, merge nums2 into nums1as one sorted array.

Note:

  • The number of elements initialized in nums1 and nums2 are m and nrespectively.
  • You may assume that nums1 has enough space (size that is equal to m+ n) to hold additional elements from nums2.

Example:

Constraints:

  • -10^9 <= nums1[i], nums2[i] <= 10^9
  • nums1.length == m + n
  • nums2.length == n

给定两个排序的整数数组nums1和nums2，将nums2合并为nums1作为一个排序的数组。在nums1和nums2中初始化的元素数分别为m和n。可以假设nums1有足够的空间（大小等于m + n）来容纳nums2中的其他元素

例：输入：nums1 = [1,2,3,0,0,0]，m = 3, nums2 = [2,5,6], n = 3, 输出：[1,2,2,3,5,6]

限制条件题目中已经给出。

解题思路一：暴力解法

nums1数组的空间是m+n，其元素数量为m个，其剩余的n个元素都为零，因此考虑将nums2数组中的元素全部加到nums1的n个为0元素的位置，这样也不需要额外的开辟内存空间。随后再对新组成的nums1进行排序。算法很简单，容易实现，但是其时间复杂度为O(n2)，因此可以考虑对算法进行优化。

解题思路二：

由于两个数组都是有序的，所有只要按顺序比较大小即可。这样就从 nums1 和 nums2 数组的末尾开始一个一个比较，把较大的数，按顺序从后往前加入混合之后的数组末尾。进行 while 循环，如果i和j都大于0，再看如果 nums1[i] > nums2[j]，说明要先把 nums1[i] 加入混合数组的末尾，加入后k和i都要自减1；反之就把 nums2[j] 加入混合数组的末尾，加入后index和j都要自减1。循环结束后，有可能i或者j还大于等于0，若j大于0，那么还需要继续循环，将 nums2 中的数字继续拷入 nums1。若是i大于等于0，那么就不用管，因为混合数组本身就放在 nums1 中。参考
