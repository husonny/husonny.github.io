（String、KMP）

[28] Implement strStr() - 实现 strStr()

Implement strStr().

Return the index of the first occurrence of needle in haystack, or -1 if needle is not part of haystack.

Example 1:

Example 2:

Clarification:

What should we return when needle is an empty string? This is a great question to ask during an interview.

For the purpose of this problem, we will return 0 when needle is an empty string. This is consistent to C’s strstr() and Java’s indexOf().

Constraints:

  • haystack and needle consist only of lowercase English characters.

实现strStr()。

返回在haystack中第一次出现的指针的索引，如果指针不是haystack的一部分，返回-1。

范例1：输入：haystack =“ hello”，needle =“ ll”, 输出2
范例2：输入：haystack =“ aaaaa”，needle =“ bba”, 输出：-1

当针头是空的时候，我们应该返回什么?在面试中这是一个很好的问题。为了解决这个问题，当needle为空字符串时，我们将返回0。这与C的strstr()和Java的indexOf()是一致的。

本题也可采用KMP算法来求解，貌似要更简单一些，由于博主并不知道KMP算法是个什么鬼，所以还是采用传统的暴力破解法。

思路：这一题和求最长前缀问题LeetCode-14.[Easy] Longest Common Prefix有一定的相似之处，求前缀问题是不知道前缀具体是什么，需要经过不断地比较才能最终判断出，可能存在最长前缀也可能不存在。而此题是给定了一个字符串needle，要求判断出该给定字符串needle是否存在于原始字符串haystack中，存在则返回具体位置的索引，不存在返回-1；

直接采用暴力破解法，先对haystack进行遍历，逐个对haystack[i]和needle[0]进行比较，若不等返回-1，若相等，则继续比较haystack[i+1]和needle[1]，只要对应元素是相等的，则一直到needle中所有字符都比较完毕完毕。判断条件：当needle[j]的索引j与其长度相等时，则说明比较完成了，这时可以直接返回i，即为所需要的索引的值。其他情况则返回-1。

本代码中的核心是如何判断haystack[i+1]和needle[1]的相等性问题，以及如何才能在满足第一个字符对应相等的情况下，保持haystack和needle同时递增，而haystack[i + j] != needle[j]能够同时解决上述问题。
